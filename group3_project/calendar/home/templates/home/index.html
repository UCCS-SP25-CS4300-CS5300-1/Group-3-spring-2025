How to use Calendai?

First, sign into Canvas, then copy the URL you get into the first box below.

Second, go to Canvas -> Account -> Settings -> Approved Integrations

Generate a new Access Token, then copy the token into the second box below.

Congratulations! You can now use Calendai.
